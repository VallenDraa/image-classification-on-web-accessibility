Kenapa Alt attribute itu penting untuk aksesibilitas?

Alt attribute berfungsi untuk mendeskripsikan sebuah gambar apabila gambar tersebut gagal ditampilkan atau ketika pengguna menggunakan Screen Reader

Jadi Alt attribute akan sangat membantu bagi penyandang disabilitas seperti tuna netra untuk menavigasikan gambar pada sebuah website dengan lebih mudah.
